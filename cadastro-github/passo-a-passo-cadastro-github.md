**Recomendações gerais:**
- Tirar **todos** os prints usando a tela cheia (em geral é a tecla F11 que faz isso, qualquer coisa google ou me pergunte), dessa forma não precisará cortar a parte das guias e tb não precisará cortar o rodapé (usuário windowns mostrando no canto inferior esquerdo a logo do windowns, nooo! kkkkkk). Tem prints dos dois tipos nesses prints. Muita gente não percebe, mas tem gente que percebe. Isso faz com que entre uma imagem e outra seja possível de se perceber a diferença no tamanho da fonte, por exemplo, e ficará ainda mais perceptivíel se usar transições numa apresentação. Usar tela cheia!

- Apague as propagandas do e-mail cobrindo-as com partes **da mesma cor**.

- **Apagar os outros e-mails deixando apenas o do github**, (para que expor seus e-mails pessoais). Basta usar algo como o Paint, mas fazer com carinho para não ficar "grosseiro".

- Usar em português tudo que for possível.

- Usar o nome padrão do print e deixar os números serem incrementados pelo próprio print, (acredito que vai facilitar identificar a ordem de qual é qual depois).
